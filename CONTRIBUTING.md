# Contributing to PersonaKit

Thanks for contributing a persona! Here's how to make sure it fits the site.

## The easy way

Go to [personakit.dev/contribute](https://personakit.dev/contribute), paste your files, fill in the details, and submit. We handle the rest.

## File format

Every persona has two files: **IDENTITY.md** and **SOUL.md**.

```markdown
What you help with:

- **Capability.** Brief description.

Optional closing line (1 sentence max).
```

### SOUL.md

```markdown
# Soul

## Personality

Short paragraph for the overall vibe (2-3 sentences). Then bullet the specific traits.

- Trait stated directly. 1-2 sentences max.
- Another trait.
- Another trait.

## Communication Style

- Concrete behavior as a bullet. No paragraphs here.
- Another behavior.
- Another behavior.

## Boundaries

- What this persona won't do. One per bullet.
- Another boundary.

## Values

- **Value name.** Why it matters.
- **Another value.** Explanation.
```

## Writing style

Write like a human, not an AI. Specifically:

- **No em dashes (â€”).** Use periods, commas, or colons.
- **No filler phrases.** Cut "it's worth noting", "at its core", "it's important to remember".
- **No AI vocabulary.** Avoid "delve", "nuanced", "meticulous", "holistic", "leverage", "tapestry", "landscape".
- **Vary sentence length.** Mix short and long. Don't make everything medium.
- **Be direct.** If the persona has a strong opinion, state it. Don't hedge.
- **Personality uses prose + bullets.** A short paragraph for voice, then bullets for individual traits.
- **Communication Style is bullets only.** Each behavior gets its own line.

## Tags

Pick 1-3 from this list:

coding, creative, support, humor, productivity, research, writing, analysis, teaching, gaming, technical, professional, casual, friendly, direct

## Limits

- 50KB max total (both files combined)
- Persona name: 1-50 characters
- Description: 1-200 characters
- Author handle: lowercase letters, numbers, hyphens only
